# Hello World

Winnie CSS is a collection of css variables and component styles that can be used to style components in any framework.

## Installation

Let's get you up and running with Winnie CSS.

### 1. Install Winnie CSS

Using the package manager of your choice, install Winnie CSS from your command line:

### 2. Import the CSS

Import the Winnie CSS file at the root of your application in the place of your choice:

### 3. Configure Winnie CSS

Configure the `body` element with Winnie CSS specific attributes

```tsx title="root.tsx" {6-9}
export default function Root() {
  return (
    <html>
      <head></head>
      <body
        class="winnie-ui"
        data-accent-color="orange"
        data-scaling="100%"
        data-radius="md"
      >
        <App />
      </body>
    </html>
  );
}
```

### 4. Start Building

Your app is now ready to go with Winnie CSS.

```css title="my-button.css"
.my-button {
  background: var(--wui-color-accent-9);
  color: var(--wui-color-accent-contrast);
  height: 32px;
  padding-inline: var(--wui-space-3);
}
```
