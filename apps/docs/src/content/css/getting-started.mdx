Winnie CSS is a collection of opinionated tools designed to help style applications in any framework. Some of the ideas and philosophies implemented in Winnie CSS were inspired by . The goal of Winnie CSS is to be an opinionated extension of your current CSS tool set. Whether you use plain old CSS, Tailwind, or any variety of CSS-in-JS, Winnie CSS provides the primitive styles you need to build the best experience for your users.
