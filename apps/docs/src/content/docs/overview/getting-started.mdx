Goho CSS is a collection of css variables and component styles that can be used to style components in any framework.

## Installation

Let's get you up and running with Goho CSS.

### 1. Install Goho CSS

Using the package manager of your choice, install Goho CSS from your command line:

{/* prettier-ignore */}
<Tabs>
  <TabItem label="npm">
    ```bash frame="none"
    npm install @goho/css
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash frame="none"
    yarn add @goho/css
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash frame="none"
    pnpm add @goho/css
    ```
  </TabItem>
  <TabItem label="bun">
    ```bash frame="none"
    bun install @goho/css
    ```
  </TabItem>
</Tabs>

### 2. Import the CSS

Import the Goho CSS file at the root of your application in the place of your choice:

{/* prettier-ignore */}
<Tabs>
  <TabItem label="React">
    ```tsx title="main.tsx"
    import "@goho/css";
    ```
  </TabItem>
  <TabItem label="CSS">
    ```css title="global.css"
    @import "@goho/css";
    ```
  </TabItem>
</Tabs>

### 3. Configure Goho CSS

Configure the `body` element with Goho CSS specific attributes

```tsx title="root.tsx" {6-9}
export default function Root() {
  return (
    <html>
      <head></head>
      <body
        class="goho"
        data-accent-color="orange"
        data-scaling="100%"
        data-radius="md"
      >
        <App />
      </body>
    </html>
  );
}
```

### 4. Start Building

Your app is now ready to go with Goho CSS.

```css title="my-button.css"
.my-button {
  background: var(--go-color-accent-9);
  color: var(--go-color-accent-contrast);
  height: 32px;
  padding-inline: var(--go-space-3);
}
```
